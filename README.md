# Лобкометр Telegram Bot

Бот для измерения лобков в Telegram групповых чатах.

## Команды
- `/lobok` - увеличить лобок (КД 15 мин, при 1000+ см - 10 мин)
- `/toplobok` - топ-30 игроков
- `/editlobok <имя>` - дать имя лобку
- `/lobokinfo` - информация о себе
- `/start` - приветствие

## Админ-команды (только @trim)
- `/rak @username` - проверить статус
- `/rak @username Yes` - выдать рак
- `/rak @username No` - вылечить

## Хостинг
Работает на Render.com 24/7
